树和图是数组和链表的延申

- 图的应用：vite和webpack中的依赖图
- 栈的应用：标签的匹配

```
[span,p,div]
<div>
  <p>
    <span></span>
  </p>
```

通过栈的先入后出的特性可以匹配出div缺少标签匹配

另外一种数组+链表的数据结构

```
{a:1,b:2}

a->hash算法转换->数字1
b->hash算法转换->数字2

插入到数组中[xxx,数字1,xxx,数字2,xxxx]

假如变成这样，出现重复
{a:1,b:2,c:1}
可以选择扩容数组，也可以将数组的每个索引下的空间变成链表
```

链表的查找可以用二分查找：时间复杂度为O(logn)
